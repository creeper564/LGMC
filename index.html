🐺 LGMC, qu'est-ce que c'est ?

LGMC est un serveur Minecraft Java en développement qui reprend le célèbre jeu de société Loup-Garou dans Minecraft.

Les maîtres du jeu pourront organiser des parties. Les joueurs seront prévenus sur Discord lorsqu'une partie sera lancée.

Chaque partie sera limitée à 10 joueurs.

🎭 Les rôles

🏹 Chasseur

Lorsqu'un Chasseur est tué pendant une partie, il a le droit de tuer une personne de la partie.

🧪 Sorcière

La Sorcière possède 2 potions pendant toute la partie :

🟢 Une potion pour sauver une personne.
🔴 Une potion pour tuer une personne choisie.

Une potion utilisée ne peut plus être réutilisée.

🔮 Voyante

Pendant la nuit, elle peut regarder la carte d'une personne qui joue.

🐺 Loup-Garou

Pendant la nuit, les Loups-Garous doivent tuer un joueur.

👧 Petite Fille

Pendant la nuit, elle doit se faire passer pour un Loup-Garou au péril de sa vie.

👨 Villageois

Pendant la nuit, les Villageois ne font rien.

💘 Cupidon

Lors de la première nuit, Cupidon choisit deux personnes parmi les joueurs.

Si l'une des deux personnes meurt, l'autre personne choisie meurt aussi.

❓ FAQ

🛏️ Java ou Bedrock ?

Java uniquement.

💰 Le serveur est-il Pay-to-Win ?

Non, tous les joueurs peuvent progresser sans payer.

🚀 Quand ouvre le serveur ?

Aucune date prévue pour le moment. L'ouverture dépendra du développement et du staff disponible.

🛠️ Comment signaler un bug ?

Ouvrez un ticket sur Discord.

🎮 Peut-on lancer une partie ?

Non, seuls les maîtres du jeu peuvent lancer une partie. Les joueurs seront ping lorsqu'une partie sera disponible.
